# LazyInit

A simple, lightweight yet easy to use init system! Works on both musl libc and gnu libc. Made with love in Kotlin!

# Philosophy

Keep speed, simplicity and usability in one. Don't trade off simplicity for usability!

# For use in:

Minimal environments
Daily use
No nonsense-crap type of systems.
# Tested on:

Alpine Linux (x86_64)
Alpine Linux (arm64)
Void Linux (musl) (arm64)
Void Linux (glibc) (arm64)
Void Linux (musl) (x86_64)
Void Linux (glibc) (arm64) If the LazyInit Project does not work on a specific Linux distro, please open a issue! Thank you :3
